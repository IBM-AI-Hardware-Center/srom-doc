srom_param_grid Contains classes related to Parameter Grid which can be used with SROMPipeline. These classes are used in performing exhaustive or randomized grid search.
srom_pipeline SROMPipeline class.
srom_single_node_search This module is used for single node based exhaustive/random search. Here, pipeline is executed with all/some parameters given in param grid for each estimator on single node depending on mode. It returns the best scores, best estimators and number of combinations.
srom_spark_search This module is used for spark based exhaustive/random search. Here, pipeline is executed with all/some parameters given in param grid for each estimator on spark cluster depending on mode.It returns the best scores, best estimators and number of combinations.
